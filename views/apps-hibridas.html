Aplicaciones híbridas

Una aplicación desarrollada como una página web que se compila dentro de un marco de aplicación nativa

WebView

Componente nativo que interpreta documentos HTML.
En otras palabras es un pequeño navegador.

Consideraciones

  • La aplicación se desarrolla como una página web, pero no es una página web.
  • Los recursos usados por el HTML deben ser almacenada dentro de la aplicación. Debe evitarse en lo posible utilizar recursos directamente desde Internet.
  • El WebView no interpreta lenguaje del servidor (PHP, Ruby, Pyhton, Node.js, etc).
  • El dinamismo de la aplicación se trabaja desde JavaScript en el FrontEnd, nunca con leuguaje de servidor.
