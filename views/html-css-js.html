HTML

HyperText Markup Language (lenguaje de marcas de hipertexto), es un lenguaje de marcado para la elaboración de páginas web, acargo de la World Wide Web Consortium (W3C). Es un dialecto de XML.

No es un lenguaje de programación.

CSS

Cascading Style Sheets (Hoja de estilo en cascada), es un lenguaje de hoja de estilos acargo de la World Wide Web Consortium (W3C), utilizado para definir la presentación de un documento estructurado. Busca separar la estructura de un documento de su presentación.

No es un lenguaje de programación.

JavaScript

Lenguaje de programación interpretado dialecto de ECMAScript, orientado a objetos, basado en prototipos, imperativo, de tipeado débil y dinámico.

Si un lenguaje de programación,
pero no se java Java.

Framework

Un framework (marco de trabajo) es conjunto especificaciones y herramientas reutilizables destinadas a resolver un tipo de problema específico.

Librerías, código reutilizable, utilidades, entre otras.

Angular JS

Framework JS MVC open-source mantenido por Google utilizado para crear páginas all-in-one. Principal función en la ampliación de HTML mendiante two data binding.

Angular Material

Framework basado en Angular JS que implementa las especificaciones de Material Design de google. Ofrece un conjunto de componentes UI reusables.
